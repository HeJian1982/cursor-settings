# Cursor AI 规则全局工作流

## 概述

这是一个 Cursor IDE 的全局 AI 工作流，可在任何项目中使用。

## 工作流程

### 1. 接收任务

接收到用户任务后：

1. 理解任务目标
2. 识别涉及的代码区域
3. 制定实施计划
4. 执行代码变更

### 2. 代码质量保证

每次代码变更后：

```
npm run typecheck  # 0 错
npm run lint       # 0 警告
```

### 3. Git 持久化

变更通过质量检查后：

```
git add -A
git commit -m "<type>(<scope>): <subject>"
git push origin main
```

### 4. 用户反馈

完成后：

- 说明做了什么
- 列出改动的文件
- 说明验证结果
- 提示待办或风险

## 适用场景

- 任何新项目初始化
- 日常功能开发
- Bug 修复
- 代码重构
- 文档编写
